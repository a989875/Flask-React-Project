# [模擬線上購物網站](https://reactflaskshoppingapp.herokuapp.com)
---

本專案為模擬線上購物網站，內容包含

1. 註冊/登入系統
2. 商品展示
3. 商品下單
4. 上傳商品

---
# Demo
Heroku連結: https://reactflaskshoppingapp.herokuapp.com
可在註冊/登入頁面建立自己的帳號，使用網站功能

---
# 使用技術
---
1. 前端: 
* ReactJS

2. 後端: 
* Python Flask
* Flask-RestFul,Flask-SQLAlchemy
